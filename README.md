# Json-Diff-Api
> Api to evaluate diffence between Json's Content

## Installation

OS X & Linux:

```sh
docker pull mongo
```

```sh
docker run -d -p 27017:27017 -p 28017:28017 -e AUTH=no mongo
```

## Usage example

```sh
mvn spring-boot:run
```

## Access in browser

```sh
http://localhost:8080/swagger-ui.html
```

## Stack
Java 8
Spring Boot 2
Lombok
Swagger
Junit & Mockito
